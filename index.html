🧲 磁力机制

N + N同极 → 排斥远离
S + S同极 → 排斥远离
N + S异极 → 互相吸引

🎯 游戏目标（对抗制）

两位玩家各自竞速，从顶部出生点出发，抵达底部金色目标点 G！
到达后消失，双方都到达后游戏结束，比较总分决胜负。
总分 = 金币数×10 + floor(600÷到达秒数)
善用磁力排斥干扰对手，磁力吸引辅助自己，传送门 T快速捷径！
按 G / . 激发磁性，临时5倍磁力！
按 H 键或点击外部关闭
